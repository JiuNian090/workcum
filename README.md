# 工作时间记录与排班管理系统

一个使用 React、Tailwind CSS 和 Vite 构建的简洁、优雅的工作时间记录与排班管理 Web 应用程序。帮助自由职业者、项目经理和团队领导轻松跟踪工作时间、管理排班并生成可视化统计。

## 项目特点

- **一体化管理平台**: 集时间记录、排班管理和统计分析于一体，无需切换多个工具
- **双重视图支持**: 提供周视图和月视图两种排班日历，满足不同场景的查看需求
- **智能数据同步**: 时间记录可自动同步到排班日历，统一管理工作安排
- **灵活班次模板**: 支持创建自定义班次模板，快速填充重复性工作安排
- **响应式设计**: 优雅的响应式界面，完美适配移动设备、平板电脑和桌面设备
- **本地数据存储**: 所有数据安全存储在浏览器本地，确保隐私和离线可用性
- **便捷数据迁移**: 支持数据导出和导入功能，便于备份、转移或在不同设备间同步

## 核心功能

### 时间记录管理
- **便捷记录**: 快速添加工作时间记录，包括开始和结束时间
- **班次模板**: 支持使用自定义班次模板快速填充记录信息
- **智能同步**: 时间记录自动同步到排班日历，统一管理工作安排
- **详细备注**: 可添加备注信息，记录工作内容或重要事项

### 排班日历管理
- **双重视图**: 提供周视图和月视图两种排班日历，满足不同场景需求
- **直观展示**: 以日历形式直观展示所有时间记录和排班安排
- **灵活编辑**: 支持在日历中直接添加、编辑和删除排班任务
- **今日高亮**: 自动高亮显示当前日期，快速定位今日安排

### 自定义班次模板
- **模板创建**: 创建和管理自定义班次模板，包含班次名称、工作时间
- **工时设置**: 支持设置自定义工时，适应不同工作安排需求
- **快速应用**: 在记录和排班中快速应用班次模板，提高操作效率
- **灵活管理**: 支持编辑和删除班次模板，随时调整工作安排

### 可视化统计分析
- **时间筛选**: 支持自定义日期范围筛选，生成特定时间段统计
- **快捷选择**: 提供"本周"和"本月"快捷日期选择功能
- **数据汇总**: 自动计算总工时，提供直观的数据汇总展示
- **Excel导出**: 支持将统计数据导出为 Excel 文件进行进一步分析

### 数据管理与迁移
- **数据导出**: 支持导出所有数据为 JSON 文件进行备份
- **数据导入**: 支持从 JSON 文件导入数据恢复信息
- **安全清除**: 提供清除所有数据功能，确保隐私安全（谨慎使用）
- **跨设备同步**: 通过导入导出功能实现不同设备间的数据同步

## 项目背景

这个项目最初是为了给我的妻子思语小姐创建一个方便计算工时的工具，现已发展成为一个功能完整的工作时间记录与排班管理系统。无论您是自由职业者、项目经理还是团队领导，这个工具都能帮助您更好地管理时间，提高工作效率。

## 快速开始

### 前置要求

- Node.js (版本 14 或更高)
- npm 或 yarn

### 安装步骤

1. 克隆仓库:
   ```bash
   git clone <repository-url>
   ```

2. 进入项目目录:
   ```bash
   cd time-tracker
   ```

3. 安装依赖:
   ```bash
   npm install
   ```

### 开发环境

启动开发服务器:

```bash
npm run dev
```

应用程序将在 `http://localhost:4000` 上可用。

### 生产构建

创建生产构建版本:

```bash
npm run build
```

构建文件将位于 `dist` 目录中。

### 部署

此应用程序可以部署到任何静态托管服务，包括:

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

#### Cloudflare Pages 部署指南

要将此应用程序部署到 Cloudflare Pages，请按照以下步骤操作：

1. **推送代码到 GitHub**：
   将您的代码推送到 GitHub 仓库（如果您还没有的话）。

2. **登录 Cloudflare Dashboard**：
   访问 [Cloudflare Dashboard](https://dash.cloudflare.com/) 并登录您的账户。

3. **创建 Pages 项目**：
   - 在左侧菜单中选择 "Workers & Pages"
   - 点击 "Create application" > "Pages" > "Connect to Git"

4. **连接到 Git 仓库**：
   - 选择您的 GitHub 账户和对应的仓库
   - 选择您要部署的分支（通常是 main 或 master）

5. **配置构建设置**：
   - **构建命令**: `npm run build`
   - **构建输出目录**: `dist`
   - **根目录**: 保持为空（默认为仓库根目录）

6. **部署**：
   点击 "Save and Deploy" 开始部署过程。

部署完成后，Cloudflare 会为您提供一个 `.pages.dev` 的临时 URL 来访问您的应用程序。您也可以配置自定义域名。

## 使用说明

1. **时间记录**: 
   - 在"记录"页面添加您的工作时间记录
   - 可选择预设的自定义班次模板快速填充记录信息
   - 支持同步时间记录到排班日历，统一管理工作安排

2. **排班管理**:
   - 在"排班"页面使用周视图或月视图查看和管理日程安排
   - 点击任意日期添加新的排班任务
   - 支持编辑和删除已有的排班记录

3. **统计分析**:
   - 在"统计"页面选择日期范围生成工时统计
   - 支持"本周"和"本月"快捷日期选择
   - 可将统计数据导出为 Excel 文件进行进一步分析

4. **自定义班次**:
   - 在"记录"页面的自定义班次管理区域创建和管理班次模板
   - 可设置班次名称、标准工作时间及自定义工时

5. **数据管理**:
   - 在"我的"页面导出所有数据为 JSON 文件进行备份
   - 支持从 JSON 文件导入数据恢复信息
   - 提供清除所有数据功能（谨慎使用）

## 技术栈

- **React 18**: 用于构建用户界面的 JavaScript 库
- **React Router v6**: 声明式路由管理，实现单页应用导航
- **Tailwind CSS**: 实用优先的 CSS 框架，实现响应式设计
- **Vite**: 新一代前端构建工具，提供快速的开发体验
- **SheetJS (xlsx)**: 用于 Excel 文件的导出功能
- **date-fns**: 现代 JavaScript 日期处理工具库
- **i18next**: 国际化框架，支持中英文多语言功能
- **LocalStorage**: 浏览器本地存储，用于数据持久化
